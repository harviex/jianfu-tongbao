# 整治形式主义为基层减负公开通报典型问题数据库

📋 本数据库收集并公开中央及各省级层面整治形式主义为基层减负专项工作机制办公室通报的典型问题，供各级干部学习对照、引以为戒。

## 访问地址

🌐 **[https://jf.uk5.uk](https://jf.uk5.uk)**

## 功能特性

- 📊 **数据统计**：实时显示通报总数、涉及省份、问题类型等关键指标
- 🔍 **智能搜索**：支持关键词全文搜索
- 🏷️ **多维筛选**：按行政层级、问题类型、涉及省份筛选
- 📄 **详情查看**：点击卡片查看通报全文内容
- 📝 **情况反映**：在线提交增加基层负担有关情况反映

## 数据来源

- 中央层面整治形式主义为基层减负专项工作机制办公室通报
- 各省级/市级整治形式主义为基层减负专项工作机制办公室通报
- 主要来源：人民网、各省纪委监委网站
